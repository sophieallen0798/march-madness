How It Works

Scoring System:

  • Round 1: 1 point per correct pick
  • Round 2: 2 points
  • Round 3 (Sweet 16): 4 points
  • Round 4 (Elite Eight): 8 points
  • Round 5 (Final Four): 16 points
  • Championship: 32 points

Tournament data is synchronized from the NCAA API by an administrator. Scores and standings are updated as games complete.
